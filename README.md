# README #

Webmoon is created for scanning website's subdomains, used only for ethical purposes


### How do I get set up? ###

* Clone The Repository
* Open Terminal
* Start The Tool (python webmoon.py -u [URL] -w [Wordlist] -b (browser pop up, optional) -i [Response Code])
* Enjoy!
